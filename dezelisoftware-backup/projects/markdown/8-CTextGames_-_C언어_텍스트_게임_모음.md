### 개요

**첫 프로젝트**다. 2017년 중학교 코딩 동아리 축제 출품작으로 만든 C언어 텍스트 게임 두 편(서바이벌 블랙잭 / 프로그램 강화 게임)과, 2024년에 첫 게임을 Python Tkinter로 다시 만든 **블랙잭 2.0.0**까지 한 레포에 묶여 있다.

### 중학교 동아리 축제

당시 중학교 동아리 축제였다. 부원 대부분이 스크래치로 작품을 만들었는데, 코딩 동아리 전시가 스크래치 일색인 게 부자연스러워 직접 C언어로 게임 두 편을 만들어 출품했다. 코드는 인터넷 검색을 발판으로 처음 짠 결과물이고, 이후 모든 개발 프로젝트의 출발점이 되었다.

### 기술 스택

- **2017 (C 콘솔 게임)**: C, Windows `<windows.h>`(gotoxy 구현용), `<stdlib.h>`(rand)
- **2024 (블랙잭 2.0.0)**: Python, Tkinter, SQLite3, Pillow, Pyglet, pygame(mixer), threading, datetime, typing

### 1. 서바이벌 블랙잭 게임 (2017, C)

블랙잭 룰을 기반으로 한 1인 서바이벌 카드 게임이다.

룰:
- 라운드마다 목표점수가 정해져 있고, 플레이어 점수는 라운드 간 누적된다.
- 입력 `1` → 카드 한 장 추가, `2` → 그만 뽑기.
- 한 라운드에서 카드 합 > 21이면 Bust → 0점.
- 카드 합 == 21이면 블랙잭 → 25~50 사이 보너스 점수 추가.
- 라운드 별 목표점수 = `2 × 라운드 × 라운드`. 누적 점수가 목표 미달이면 게임오버.

원래 의도는 AI 또는 2인 플레이가 가능한 블랙잭이었지만, 당시 능력 안에서는 구현이 불가능해 1인 서바이벌 형태로 마무리됐다.

### 2. 프로그램 강화 게임 (2017, C)

블랙잭 다음으로 만든 두 번째 게임. 기본금 1000원으로 프로그램을 사고 강화해 등급을 올리는 시스템이다.

기능 4가지:

- **프로그램 강화**: 성공 확률 = `(100 - 현재 등급)%`. 성공 시 +1 등급, 실패 시 -5 등급.
- **프로그램 사기**: 등급별 가격 (1등급 100원 / 10등급 300원 / 20등급 1000원 / 30등급 3000원 / 30~60 랜덤 등급 10000원).
- **아이템 상점**: 강화 확률 +2/+5/+10% (1000/2000/3500원), 강화 성공 시 추가 +3 등급(5000원) / +5 등급(10000원).
- **로또**: 지불 금액에 대해 10% 확률로 2/3/5/10배 당첨, 50% 확률로 전액 손실.

승리 조건은 없고, 등급 100 이상부터는 아이템 사용이 강제된다. 자기 한계까지 강화 등급을 끌어올리는 게 목적인 게임이다.

#### 이스터에그: "2017"

메뉴에서 `1/2/3/4` 외 입력 시 보통 "다시 선택해 주세요"가 뜨지만, 제작 연도인 **`2017`**을 입력하면 30000원이 지급된다. 본래는 강화 시스템 테스트 편의를 위해 넣은 코드인데, 동아리 축제에 들른 친한 친구들에게는 슬쩍 알려주기도 했다.

### 3. 화면 처리 — gotoxy 함수

C 콘솔 게임의 핵심은 화면 갱신이다. 2017 버전은 Windows API `SetConsoleCursorPosition`을 래핑한 `gotoxy(x, y)` 함수로 콘솔 커서를 좌표로 이동시키며 텍스트만으로 화면 전환을 만들었다 — 그 시기 C 콘솔 게임의 정석 패턴이다.

랜덤 요소(블랙잭 카드, 강화 확률, 로또 등)는 모두 `<stdlib.h>`의 `rand()`로 처리.

### 4. 서바이벌 블랙잭 2.0.0 — Python Tkinter 리뉴얼 (2024)

2024년 2월, 첫 게임이었던 블랙잭을 Python Tkinter 기반으로 다시 만들었다. C 콘솔 형식이 너무 오래된 느낌이었고, 첫 프로젝트인 만큼 같은 룰을 현재의 도구로 한 번 다시 구현하기 위한 리뉴얼이다.

룰은 2017 버전과 동일. 차이는 다음과 같다.

- **GUI**: Tkinter로 카드/메뉴/배경 통합
- **랭킹 저장**: SQLite3에 점수와 플레이 날짜 저장
- **음악/효과음**: pygame.mixer로 BGM, threading으로 메인 루프와 분리
- **이미지/폰트**: Pillow + Pyglet으로 카드 이미지와 커스텀 폰트 처리

### 메모

- 본인의 코딩 시작점에 해당하는 레포다. 2017년 C 콘솔 게임 두 편과 2024년 Tkinter 리뉴얼이 한 자리에 묶여 있는 이유도 그것이다.
- 2017 코드는 당시 모습 그대로 보존되어 있어, 문법이나 구조 정리는 거의 들어가지 않았다.
